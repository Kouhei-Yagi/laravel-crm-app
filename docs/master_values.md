# マスタ値一覧（CRM システム）

本ドキュメントは、CRM システムで使用するマスタ値とその意味をまとめたものです。
UI のプルダウン、バリデーション、業務ルールの基準として使用します。

---

## 1. 顧客ステータス（status）

-   **見込み**：まだ取引前の見込み顧客
-   **商談中**：商談が進行中
-   **成約**：契約が成立した顧客
-   **失注**：契約に至らなかった顧客
-   **休眠**：長期間連絡が取れていない顧客

---

## 2. 顧客ランク（rank）

-   **A**：重要顧客（売上・頻度が高い）
-   **B**：通常顧客
-   **C**：優先度が低めの顧客

---

## 3. 案件ステータス（project_status）

-   **見積中**：見積を提示している段階
-   **提案中**：提案書を提出している段階
-   **契約済**：契約が成立した案件
-   **失注**：契約に至らなかった案件

---

## 4. 対応種別（interaction_type）

-   **電話**：電話での対応
-   **メール**：メールでの対応
-   **訪問**：顧客先を訪問して対応
-   **打合せ**：会議・打合せでの対応
